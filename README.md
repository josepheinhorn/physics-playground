> **The Problem:** Fine-tuning the "feel" of a platformer—gravity, friction, and jump height—requires constant code changes and reloads.
> **The Solution:** A real-time sandbox where you can manipulate physics constants via a UI and see the immediate impact on your character's movement.

### 🚀 Live Demo
**[→ Launch Physics Playground ←](https://josepheinhorn.github.io/physics-playground/)**

---

### ⚡ Key Features
* **Real-Time Tuning:** Sliders for Gravity, Acceleration, Friction, and Jump Impulse.
* **Interactive Testing:** Full WASD/Arrow Key support to control the character within the sandbox.
* **Live Export:** Once the movement feels right, click "Export Config" to get a JSON snippet ready for the main game engine.
* **Collision Debugging:** Visual bounds showing the character's hitboxes against the "floor."

### 📖 How to Use
1. Open the [Live Demo](https://josepheinhorn.github.io/physics-playground/).
2. Use the sliders on the left to adjust how heavy or floaty the character feels.
3. Move and jump around the stage to test the "game feel."
4. Copy the JSON output to save your perfect physics constants.
